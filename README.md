# Análisis sectorial del S&P 500
Este código en Python permite analizar el comportamiento y rendimientos de los sectores del S&amp;P 500 o de acciones individuales, con la opción de calcular correlaciones, indicadores financieros (Sharpe y Sortino Ratio) e indicadores técnicos (SMA y EMA). Las librerías utilizadas fueron yfinance, matplotlib, plotly, pandas y numpy.
